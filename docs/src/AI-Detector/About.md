# AI Detector

AI Detector is a lightweight, fast, and user‑friendly tool for checking whether a piece of text was likely generated by artificial generative intelligence models. Built entirely in Scratch and enhanced for TurboWarp and CodeTorch, it provides instant analysis, clear explanations, and a polished interface designed for both desktop and mobile users.

![A screenshot of the main AI Detector interface. A "Press to enter text" button lies in the middle and a toggle switch is positioned at the bottom right set to "US English".](src/AI-Detector/mainscreenshot.png)

The UI is clean, readable, and optimized for quick text entry and analysis.
