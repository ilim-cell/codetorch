# Features - AI Detector

- ⚙️ AI Detection Engine: Quickly determines whether text appears human‑written or AI‑generated. 
- ⌨️ Simple Text Input: Type or paste text directly into the interface for instant analysis. 
- ℹ️ Detailed Explanations: Press the WHY? button after analysis to see exactly why the detector classified the text the way it did. 
- 🌐 US English Mode: If you write in US English, enable the toggle in the bottom‑right corner for improved accuracy.
- 📱 Mobile Friendly: Fully compatible with touch devices and mobile keyboards. 
- 🚀 Optimized for CodeTorch: Runs smoothly with full functionality in TurboWarp and CodeTorch environments.
