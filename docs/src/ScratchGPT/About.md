# ✨ ScratchGPT | A smart chatbot UI in CodeTorch

## Overview
ScratchGPT is a fully‑featured chatbot interface built in Scratch and enhanced for CodeTorch. It supports fullkeyboard input, caret control, mobile typing, voice mode, and a polished UI, all running inside a Scratch‑compatible environment.

- **Best experience:** On CodeTorch, your project will automatically take advantage of the TurboWarp compiler as well as other features that help with the project such as full keyboard input and reduced glitches.
- **Infinitely extensible:** The project is designed to be infinitely extensible with modular sprites, object-oriented programming coding conventions and an architecture for chatbot responses that can be customised easily.
- **Made by professionals:** All code is human written, not AI-generated. We also ensure all projects are premium designed for the easiest user experience.
