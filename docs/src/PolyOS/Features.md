## Features - PolyOS 7

- BIOS layer and kernel-driver simulations
- Vara Assistant, an intelligent assistant that can help you with tasks and answer questions
- Modern UI with a simple and clean design
- Multi-window support, allowing you to run multiple apps at once
- Customizable settings and preferences, allowing you to personalize your experience
- Built-in apps, including a file manager, media player, and setting manager
- Terminal emulator for advanced users and developers
- Disk encryption and security features to protect your data
- Custom apps for PolyOS with Xappwindow 3.0 - [*Build your own app easily with a smooth user experience*](https://codetorch.net/en/projects/106363)
- Open-source and actively maintained
